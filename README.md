# 🛸 Ultraviolet Siege

**Ultraviolet Siege** is a fast-paced, skill-based top-down arcade survival shooter. As the final line of defense against a full-scale alien invasion, you must rely on quick reflexes, strategic positioning, and adaptive weapon switching to survive the escalating extraterrestrial onslaught.

---

## 🕹️ Gameplay & Objective

The player navigates a top-down battlefield against relentless, continuous waves of alien invaders. As time progresses, the enemy presence intensifies—requiring faster reactions and smarter movement.

- 🏆 **The Win Condition:** Successfully defeat **100 alien invaders** to complete the mission and secure victory.
- 💀 **The Lose Condition:** Survival is not guaranteed. If your health pool drops to zero, the game ends immediately.

---

## 🎮 Controls

The control scheme is designed to be highly intuitive, keeping the focus entirely on fast-paced, fluid arcade action:

| Action | Control |
| :--- | :--- |
| **Movement** | `W` `A` `S` `D` keys |
| **Attack / Shoot** | `Left-Click` |
| **Switch Weapon** | `Right-Click` |

---

## ⚔️ Key Features

- **Dynamic Weapon Switching:** Cycle instantly between **three distinct weapons**. Each weapon offers a unique playstyle tailored for different combat scenarios—whether you need close-range crowd control, high-damage ranged attacks, or balanced combat efficiency.
- **Escalating Difficulty:** Enemy waves scale dynamically over time, pushing your tactical adaptability and reflexes to the limit as you edge closer to the 100-kill goal.
- **Retro Arcade Aesthetic:** Visuals are entirely handcrafted in classic, vibrant pixel art to evoke a nostalgic yet fresh retro-arcade energy.

## 🚀 How to Play / Run Locally

If you'd like to run the project from source or try the build:

1. Clone this repository:

   ```bash
   git clone https://github.com/YOUR-USERNAME/ultraviolet-siege-top-down-shooter.git
   ```

2. Open the project file inside GameMaker.
3. Press `F5` to compile and run the game.
